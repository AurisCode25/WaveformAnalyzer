WAVEFORM ANALYZER

This is the final build for the waveform analyzer. 

To run the program successfully open the folder that contains the programs executable and csv file(make sure the csv file you wish to use is in the same folderas the .exe) in powershell. Run the command in Powershell: 

        ./waveform_analyser "CSV_Filename".csv

An output.txt file will appear in that folder with all the information and will tell you if the data is compliant or not.
